Parameters separator Optional. Specifies a string to separate each element of the array. The separator is converted to a string if necessary. If omitted, the array elements are separated with a comma. If separator is an empty string, all elements are joined without any characters in between them. DescriptionEDIT The string conversions of all array elements are joined into one string. If an element is undefined or null, it is converted to the empty string. ExamplesEDIT Joining an array four different ways The following example creates an array, a, with three elements, then joins the array four times: using the default separator, then a comma and a space, then a plus and an empty string.
